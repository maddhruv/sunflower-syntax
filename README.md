### Getting Started
#### Installation
##### apm
```shell
apm install sunflower-syntax
```

##### Atom
1. Open *Settings* > *Install*
2. Set the filter to Themes
3. Search for ```sunflower-syntax``` and install
4. Open *Settings* > *Themes*
5. Select ```sunflower``` from the Syntax Theme drop-down menu
